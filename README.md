# 🪙 Conversor de Monedas en Java

Este es un proyecto desarrollado para el desafío del programa Oracle Next Education (ONE) y Alura Latam. Es una aplicación de consola en Java que realiza conversiones de divisas en tiempo real utilizando la Exchange Rate API.

## 🚀 Características
* Conversión precisa utilizando tasas de cambio actualizadas al instante.
* Soporte para múltiples monedas (USD, ARS, BRL, EUR, PYG, COP).
* Menú interactivo en la consola.
* Historial de conversiones con registro de fecha y hora exacta.

## 🛠️ Tecnologías Utilizadas
* Java 17+
* Librería Gson (para parseo de JSON)
* HttpClient nativo de Java
* Entorno: IntelliJ IDEA

## ⚙️ Cómo ejecutar este proyecto
1. Clona este repositorio en tu máquina local.
2. Ingresa a la página de [Exchange Rate API](https://www.exchangerate-api.com/) y obtén tu propia API Key gratuita.
3. Abre el archivo `Principal.java` y reemplaza el texto `"TU_API_KEY_AQUI"` con tu clave real.
4. Asegúrate de tener la librería Gson agregada a las dependencias de tu proyecto.
5. Ejecuta la clase `Principal`.
